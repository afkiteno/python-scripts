# Python-Scripts

A collection of small Python scripts and random things I've made over the years.  
Mostly for personal use, backup, and fun projects.

## Folder structure

### TF2
- `unc.py` – Python script to fetch TF2 server stats, specifically Uncletopia's US-E  
- `autoexec.cfg` – TF2 configuration file for personal setup

### P2
- `png.py` – Converts JPG images to PNG in the `png/` folder  
- `png/` – Folder for image files
- `text.py` – Python script to emulate textexpanding
- `abbreviations.json` – Cuts of mapping for `text.py`

### Random
- `pres.py` – Converts PPTX file to PDF file

## Usage Notes

- Most scripts use relative paths, so moving the repo won't break.  
- Empty folders are kept with `.gitkeep` files to ensure Git tracks them.  
- TF2 folder contains both Python scripts and game config files for convenience.
